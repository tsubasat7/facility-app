# 運命の六属性 診断ツール（内部・占い師用）

このリポジトリは占い師が対面で使う内部診断ツールです。
本番では非公開で運用してください。

## 使い方（ローカル／Pages）
1. `index.html` をブラウザで開く（または GitHub Pages にデプロイ）
2. `data/` にある JSON を編集すると診断文章を更新できます
3. 操作は占い師が行い、出力を読み上げまたはコピーして提供してください

## ファイル構成
- `index.html` - 占い師向けのUI
- `data/` - 各種テンプレ（flattype_traits.json / story.json / compatibility.json / group.json）
- `docs/manual.md` - 占い師向けマニュアル（詳細）

## 運用上の注意
- PII（個人識別情報）は保存しないでください
- 医療診断を目的に使わないでください（補助用途のみ）
