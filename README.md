# 運命の六属性診断（内部ツール）

本リポジトリは「占い師が使用する内部診断ツール」です。
来客が触れることはなく、アウトプット結果のみを提示します。

## 構成
- index.html … 完全統合ビューア
- data/*.json … 診断ロジック・テンプレ群（60命数完全対応）

## 含まれる診断
- 六属性診断
- 命数（1〜60）＋守護星座（金銀）
- 性格・本質
- 過去・育ちの傾向
- 行動タイプ
- 相性診断
- グループ診断
- 手相補正
- 姓名判断補正

## 使い方
index.html をブラウザで開き、
占い師が内部で person 情報を入力して使用します。
